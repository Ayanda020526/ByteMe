# ByteMe - Smart Quiz Learning Platform
Smart Quiz is a cross platform web and mobile based learning platform that transforms traditional studying into an interactive experience. It features adaptive quizzes, AI assisted question generation, and detailed performance analytics to help students identify weak areas and improve efficiently. 

##  Description
The system helps students identify weak areas, track their progress, and generate quizzes from study material using AI. It promotes active learning and personalized study experiences.

##  Features
- User authentication (Login/Register)
- Interactive quizzes
- Adaptive difficulty
- AI-generated questions
- Performance analytics
- Web and mobile access

##  Technologies
- Frontend: React (Web), Flutter (Mobile)
- Backend: Node.js (Express)
- Database: MySQL/PostgreSQL
- AI Integration
