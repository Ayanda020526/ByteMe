# User Stories

- As a student, I want to register an account so that I can access quizzes.
- As a user, I want to log in so that I can continue my progress.
- As a student, I want to take quizzes so that I can test my knowledge.
- As a student, I want to see my results so that I can identify my weak areas.
- As a student, I want to track my performance so that I can improve over time.

- As a lecturer, I want to create quizzes so that students can practice.
- As a lecturer, I want to view student performance so that I can identify learning gaps.

- As a user, I want the quiz difficulty to adapt so that I am appropriately challenged.
